# Replication Study: Banerjee et al. (2015)
### "The Miracle of Microfinance? Evidence from a Randomized Evaluation"
**Author:** Shine Bhatta  
**Institution:** Kathmandu University, Department of Economics  
**Date Started:** March 2025  

---

## Overview
This repository contains a replication of Banerjee, Duflo, Glennerster & Kinnan (2015), 
published in the American Economic Journal: Applied Economics. The paper evaluates the 
impact of microcredit access on household welfare outcomes using a randomized controlled 
trial across 104 slum neighbourhoods in Hyderabad, India.

This project additionally extends the analysis to the Nepali context using the Nepal 
Living Standards Survey (NLSS IV, 2022/23), examining whether patterns observed in 
Hydera

## Data
Raw data is not included in this repository due to file size constraints.  
Download the following files and place them in `data/raw/`:

## Repository Structure
```
banerjee2015-replication-nepal/
├── data/
│   └── clean/          # processed data files (generated by scripts)
├── scripts/
│   ├── 00_master.R     # runs all scripts in order
│   ├── 01_load_data.R  # loads and inspects raw data
│   ├── 02_balance.R    # Table 1 — balance check
│   ├── 03_takeup.R     # Table 2 — microcredit takeup
│   ├── 04_business.R   # Table 4 — business investment
│   ├── 05_consumption.R# Table 5 — consumption outcomes
│   ├── 06_social.R     # Tables 7-8 — health, education, empowerment
│   └── 07_nepal.R      # Nepal extension — NLSS IV analysis
├── output/
│   ├── tables/         # regression output tables
│   └── figures/        # plots and visualisations
├── notes/              # annotation notes and research log
├── .gitignore
└── README.md
```
